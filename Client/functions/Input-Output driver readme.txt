To let MATLAB use LTP i/o port:
1. Download io64.mexw64 (http://apps.usd.edu/coglab/psyc770/misc/x64/io64.mexw64)
2. Place it in current MATLAB dir
3. Download self-installing system driver named inpoutx64.dll (http://apps.usd.edu/coglab/psyc770/misc/x64/inpoutx64.dll)
4. Move it to the C:\windows\system32 directory
5. Check the Microsoft Visual C++ 2005 SP1 Redistributable (x64) Package installed
6. If not: http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=18471
7. More information here: http://apps.usd.edu/coglab/psyc770/IO64.html
